# Overview

This project provides population density data for Bangladesh along with analysis and visualization tools.

# Script Reference

The scripts included in this repository are:
- `data_cleaning.py`: A script for cleaning the raw data.
- `data_analysis.py`: Contains functions to analyze the cleaned data.
- `visualization.py`: Provides functions to visualize the density maps.

# Installation

To install the required packages, run the following command:

```bash
pip install -r requirements.txt
```

# Usage

After installing the required packages, run the analysis script:

```bash
python data_analysis.py
```
